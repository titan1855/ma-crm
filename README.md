# 我的名單管理系統

美安事業潛在客戶管理 PWA App

## 功能
- 潛在客戶名單管理（姓名、電話、Email、狀態）
- FORMHD 資料記錄（Family / Occupation / Recreation / Money / Health / Dream）
- 會面記錄（會面前後 Leader 討論、本次進度、對方反應、下次約會）
- 約會行程月曆／週曆
- 優惠顧客購物記錄
- 資料完全存在手機本地（offline 可用）

## 使用方式
1. 用手機瀏覽器開啟網址
2. 點「加入主畫面」
3. 即可像 App 一樣使用，完全離線

## 部署
此專案透過 GitHub Pages 部署，網址格式：
`https://你的帳號.github.io/你的repository名稱`
